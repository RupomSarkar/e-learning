# E-Learning Website using HTML5, CSS3, Bootstrap5 and JavaScript 

![E-Learning Website](preview.png)

The project involves the creation of an e-learning website using HTML, CSS, Bootstrap 5, and JavaScript. The website aims to provide a user-friendly and responsive platform for learners to access educational content. 


## Introduction

Code Mystics is an e-learning website designed to provide a user-friendly and responsive platform for learners to access educational content. The project incorporates HTML, CSS, Bootstrap 5, and JavaScript to create an interactive and visually appealing learning experience.

## Features

- Responsive design using Bootstrap 5.
- Structured course catalog with detailed descriptions.
- Interactive lessons.
- User can register and login.
- User authentication for personalized learning experiences.
- Instructor application form.
- User can also contact us.
